How to Get API Keys

  1. Go to OpenAI API Keys

  2. Sign in or create an account
  3. Click "Create Key"
  4. Copy the key (starts with "sk-ant-")

  1. Go to Groq Console
  2. Sign in or create an account
  3. Click "Create API Key"
  4. Copy the key (starts with "gsk_")
